# Interactive Grid Duplicate Validator (Oracle APEX)

## 📌 Description
This plugin helps validate duplicate entries in Oracle APEX Interactive Grid.

## 🚀 Features
- Prevent duplicate rows
- Real-time validation
- Easy integration

## 🛠️ Technologies Used
- Oracle APEX
- JavaScript
- SQL

## 📂 Installation
1. Import plugin into APEX
2. Attach to Interactive Grid
3. Configure settings
